# Reference paper templates

These six paper template photographs are included from:

- Project: `huadeng863/handwriting-font-conversion`
- Source: https://gitee.com/huadeng863/handwriting-font-conversion/tree/master/background
- Original files: `background_01.jpg` through `background_06.jpg`
- Integration date: 2026-07-17
- Permission: the project user confirmed that the rights holder authorized inclusion in this open-source package, with source attribution.

The files were renamed descriptively for HandDraft. `reference-blank-warm.jpg`,
`reference-blank-cool.jpg`, and `reference-blank-mono.jpg` are color-treatment
derivatives of the authorized `reference-blank-desk.jpg` photograph.

The `previews` directory contains resized derivatives used only by the browser editor. Full-resolution source photographs remain the rendering inputs for PNG, PDF, and ZIP export.
